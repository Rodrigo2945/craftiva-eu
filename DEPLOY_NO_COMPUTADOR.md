# 🚀 CRAFTIVA.EU - DEPLOY FINAL (2 COMANDOS)

**Tempo total: 2 minutos**

---

## PASSO 1: Extrair

Extrai o ZIP `craftiva-eu-FINAL.zip` para uma pasta qualquer.

---

## PASSO 2: Abrir Terminal

**Mac**: Command+Espaço → escreve "Terminal"
**Windows**: Tecla Windows → escreve "cmd"
**Linux**: Ctrl+Alt+T

---

## PASSO 3: Navegar para a pasta

No terminal, escreve:

```bash
cd /caminho/onde/extraiste/craftiva-eu-FINAL
```

(Arrasta a pasta para o terminal se não souberes o caminho)

---

## PASSO 4: Executar Deploy (2 COMANDOS)

### Comando 1: Login Firebase
```bash
firebase login
```

Vai abrir o browser → Faz login → Volta ao terminal

### Comando 2: Deploy
```bash
firebase deploy
```

⏳ Aguarda 1-2 minutos...

---

## ✅ PRONTO!

Vais ver:

```
✔  Deploy complete!

Hosting URL: https://craftiva-eu.web.app
```

🎉 **O site está ONLINE!**
